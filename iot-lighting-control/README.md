# IoT照明制御プロジェクト

Raspberry Pi 3と赤外線を使って照明を制御するプロジェクト

## 環境

- Raspberry Pi 3
- SSH接続: `ssh raspi`

## 購入部品

| 部品名 | 型番 | 価格 | 数量 |
|--------|------|------|------|
| 赤外線LED 940nm | OSI5FU5111C-40 | 100円 | 5個入 |
| 赤外線受信モジュール | OSRB38C9AA | 100円 | 2個入 |
| NPNトランジスタ | 2SC1815-Y | 80円 | 10個入 |

## 回路構成

```
GPIO Pin --> 抵抗 --> 2SC1815(Base)
                         |
                      (Collector) --> 赤外線LED --> 抵抗 --> 3.3V/5V
                         |
                       (Emitter) --> GND
```
